    def feasts(season : String = "", rank_type : String = "all", datacenter : String = "Chaos") : Dataclasses::FeastResponse #

    Get information about Feasts Ranking Leaderboards. Response will be empty if there is currently no active season.

    season is the string value of the season number to get results for. Defaults to the current season by sending an empty value.

    rank_type can be the string "all", or strings of numbers from "1" to "5" inclusive. Defaults to "all".

    datacenter is the Datacenter to fetch results from. Defaults to "Chaos" as that's the default for the Lodestone page.

    def heaven_on_high(ranking_type : String = "party", job_name : String = "pld", datacenter : String = "Chaos") : Dataclasses::DeepDungeonResponse #

    Get information about Heaven on High Leaderboards.

    ranking_type is either "solo" or "party". Defaults to "party". job_name is the acronym for a job to get rankings for, for "solo" type rankings. Defaults to "pld".

    datacenter is the Datacenter to fetch results from. Defaults to "Chaos" as that's the default for the Lodestone page.

    def lodestone : Dataclasses::Lodestone #

    Retrieve all of the current Lodestone information in a single request. This endpoint is cached every 15 minutes in XIVAPI. It is highly recommended to use this method as it contains the majority of the Lodestone data and returns instantly.
