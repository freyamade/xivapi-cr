    def market(item_id : UInt64, servers : Array(String) = [] of String, datacenter : String = "", max_history : Int32 = 50) : Hash(String, Dataclasses::MarketItem) #

    Retrieve the details of a specific item on multiple servers. Specify a max_history to limit the number of historic records returned for the item, defaults to 50. If datacenter is specified, it takes priority over the servers array, and will return data for all of the servers in the chosen datacenter.

    The response from this method is a Hash of server name to the item's details.

    def market(item_ids : Array(UInt64), servers : Array(String) = [] of String, datacenter : String = "", max_history : Int32 = 50) : Array(Hash(String, Dataclasses::MarketItem)) #

    Retrieve the details of an array of items on multiple servers. Specify a max_history to limit the number of historic records returned for the item, defaults to 50. If datacenter is specified, it takes priority over the servers array, and will return data for all of the servers in the chosen datacenter.

    The response from this method is an Array of Hashes from server names to item details. The items in the array match the items in the array of item ids.
